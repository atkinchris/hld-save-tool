Hyper Light Drifter - Save Tool

This tool allows you to edit save files, to move them from one computer to another. Thank you to Heart Machine for a brilliant game!

Quickly thrown together by Chris Atkin, Source on GitHub.

This is your original save file, to be transferred to the target computer.

Select any save from the computer you are transferring your save to.

Save File Locations

These are the default locations where save files are stored. Save files are named HyperLight_RecordOfTheDrifter_0.sav, where the number is the slot of the save file.

OSX
~/Library/Application Support/com.HeartMachine.HyperLightDrifter
Windows
%LOCALAPPDATA%\HyperLightDrifter
Linux
~/.config/HyperLightDrifter
